# free-marketplace
An free market plave

Back-END:
Node.js+Express+Mysql+JWT 

Front-end:
React.js+Bootstrap+FontAwesome

How to install 
1) Crate a data base 'market_place'
2) Populate databse Using the file 'market_place.sql'
3) Download via zip or gir clone
4) cd ./back-end/ & nodemon start
5) cd ./MARKET/ & npx create-react-app front
6) Copy all content of ./front-end to ./front
7) npm i '@fortawesome/free-solid-svg-icons' --save
8) cd ./front/ & npm start
9) Enjoy!
